This project aims to create a purely vision-based autonomous navigation system. The aim of this project is to develop a mobile platform equipped with an RGB-D camera and lidar to aid people in locating their lost belongings within confined spaces. The system also comes with an intuitive GUI and a command prompt for user to interact with.

Project

Reinforcement Learning for Autonomous Vehicles with CARLA Simulator

This project develops a reinforcement learning agent to control an autonomous vehicle within the CARLA simulator. Using a Double Deep Q-Network (DDQN) enhanced with a CNN for visual processing and an LSTM for understanding motion, the agent was trained to navigate complex urban environments.

Keywords: Robotics RL CARLA
Reinforcement Learning for AVs with CARLA

Project

Autonomous Raspberry Pi Car

This project presents a complete autonomous navigation system implemented on a Raspberry Pi-powered carand demonstrates how an embeded computer with limited computation power and sensors can be leveraged to create a miniature self-driving vehicle that can detect obstacles, recognize stop signs, dynamically remap its environment, and plot alternative routes to reach target destinations.

Keywords: Robotics Self-Driving
Autonomous Raspberry Pi Car
